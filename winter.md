### Chapter 0
- Let *f, g: ℤ<sup>+</sup> → ℝ<sup>+</sup>*. Say *f = O(g)* if there is a constant *c > 0* s.t. *f(n) ≤ c·g(n)*.
- *f = Ω(g)* means *g = O(f)*
- *f = Θ(g)* means *f = O(g)* and *f = Ω(g)*
- *g(n) = 1 + c + ... + c<sup>n</sup>*
  - *Θ(1)* if *c < 1*
  - *Θ(n)* if *c = 1*
  - *Θ(c<sup>n</sup>)* if *c > 1*
